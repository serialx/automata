Complexity
----------
Input length N, Transition function size T에 대하여 Worst-case Complexity는 O(N*T) 입니다. Transition Function에 대하여 Hash Table을 사용하면 O(N)에 바운드 시킬 수 있을 것으로 보입니다. 하지만 코드의 간결성과 가독성을 위하여 최소한의 코드로 작성하였습니다.

Input Method
------------
Functional language에서 많이 사용하는 스타일을 빌려 input_dfa.txt를 eval함수를 사용하여 로드하도록 하였습니다. Set 문법인 {a, b, c, ...} 를 사용하기 위하여 Python 2.7을 최소 요구사항으로 체택하였는데, 프로젝트 입력의 간결성을 위한 선택이었습니다.
